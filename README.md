# foxid
Microscopic ID generator for Node.js.

 * URL-friendly
 * Incredibly simple
 * Good-enough performance
 * Cryptographically secure

```javascript
var foxid = require('foxid')
foxid() // "w9q567s7qj4bmp9h"
foxid(32) // "tw24cymw3kb225mctevc3csna241pnm2"
```
